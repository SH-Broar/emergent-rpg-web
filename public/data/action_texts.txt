# 대기 (wait)
# ============================================================

[wait]
1 = 잠시 걸음을 멈추고 주변을 둘러본다. 바람이 지나가고, 시간이 천천히 흐른다.
2 = 아무것도 하지 않고 서 있는 것도 때로는 필요한 일이다. 마음이 조금 가라앉는다.
3 = 하늘을 올려다보며 숨을 깊이 들이쉰다. 오늘 하루가 조용히 흘러가고 있다.
4 = 멍하니 주변을 바라본다. 평범한 풍경 속에서 새삼 작은 것들이 눈에 들어온다.
5 = 발밑의 돌 하나, 바람에 흔들리는 풀잎. 서두르지 않아도 세계는 이렇게 움직이고 있다.

[wait.Lake]
1 = 호수의 수면을 물끄러미 바라본다. 잔잔한 물결이 마음까지 고요하게 만든다.
2 = 갈대 사이로 불어오는 바람이 시원하다. 물 위에 비친 하늘이 아름답다.
3 = 물가에 앉아 수면에 손을 담근다. 차가운 물이 손끝을 적시고, 생각이 맑아진다.

[wait.Tavern]
1 = 주점 구석에 앉아 오가는 사람들을 지켜본다. 웃음소리와 잔 부딪히는 소리가 섞인다.
2 = 맥주잔 너머로 모험가들의 이야기가 들려온다. 누군가의 무용담이 흥미롭다.
3 = 나무 의자에 등을 기대고 한숨 돌린다. 주점의 온기가 피로를 녹인다.

[wait.카엘]
1 = 카엘이 옆에서 발굽으로 땅을 가볍게 두드리며 말했다. 「가만히 있는 건 성격에 안 맞는데… 뭐, 같이 있으니까 참을게.」
2 = 카엘이 갈기를 쓸어넘기며 멀리를 바라본다. 「초원의 바람이 그리울 때가 있어. 이런 날 특히.」

[wait.세리나]
1 = 세리나가 조용히 옆에 서서 같은 방향을 바라본다. 「가끔은 이렇게 쉬어가는 것도 좋죠.」
2 = 세리나가 작은 수첩을 꺼내 뭔가를 적다가, 고개를 들어 미소 짓는다. 「오늘은 평화로운 날이네요.」

# 이동 (move)
# ============================================================

[move]
1 = 발걸음을 옮긴다. 길 위의 풍경이 천천히 바뀌어 간다.
2 = 새로운 길을 걷는다. 발밑으로 돌과 흙의 감촉이 전해진다.
3 = 목적지를 향해 걸어간다. 주변의 소리가 걸음에 맞춰 바뀐다.
4 = 한 걸음씩 나아간다. 풍경이 뒤로 밀려나고, 새로운 공기가 얼굴을 스친다.

[move.카엘]
1 = 카엘이 보폭을 맞춰 나란히 걷는다. 「같이 걸으면 길이 짧게 느껴져.」
2 = 카엘의 발굽 소리가 경쾌하게 울린다. 「이 길, 전에 한번 달려봤는데 꽤 좋았어!」

[move.이리엘]
1 = 이리엘이 가볍게 날갯짓하며 앞서간다. 「이쪽으로 가면 바람이 좋은 곳이 있어요!」
2 = 이리엘이 길가의 풀을 살피며 걷는다. 「여기 약초가 자랄 것 같은데… 나중에 다시 와야겠다.」

[move.드라카]
1 = 드라카가 코를 킁킁거리며 걷는다. 「이 방향에서 맛있는 냄새가 나는 것 같은데.」
2 = 드라카의 무거운 발소리가 뒤에서 따라온다. 「300년 전에도 이 길을 걸었지. 그때도 풍경은 이랬어.」

# ============================================================
# 살펴보기 (look)
# ============================================================

[look]
1 = 주변을 천천히 둘러본다. 익숙한 풍경 속에서 어제와 다른 것은 없는지 살핀다.
2 = 눈을 가늘게 뜨고 사방을 살핀다. 이 장소에 대해 조금 더 알게 된 기분이다.
3 = 발을 멈추고 이곳의 분위기를 음미한다. 소리, 냄새, 온기 — 감각이 깨어난다.

# ============================================================
# 대화 (talk)
# ============================================================

[talk]
1 = 이야기를 나누었다. 상대의 목소리에서 살아있는 사람의 온기가 느껴진다.
2 = 대화가 끝난 뒤에도 남은 말의 여운이 가슴에 맴돈다.
3 = 사소한 대화였지만, 이 마을에 조금 더 가까워진 기분이다.
4 = 말을 주고받는 사이, 서로에 대해 몰랐던 한 면을 엿보았다.
5 = 웃음이 오갔다. 작은 대화 하나가 하루를 따뜻하게 만들기도 한다.

# ============================================================
# 거래 (trade)
# ============================================================

[trade]
1 = 물건과 동전이 손을 바꿨다. 거래란 결국 서로의 필요가 만나는 일이다.
2 = 흥정을 마치고 물건을 챙긴다. 좋은 거래였는지는 시간이 말해줄 것이다.
3 = 상인의 능숙한 손놀림을 보며 물건을 받았다. 시장의 활기가 귀에 감긴다.

[trade.Market_Square]
1 = 시장의 소란 속에서 거래를 마쳤다. 온갖 물건의 냄새와 흥정 소리가 뒤섞인다.
2 = 좌판 위에 늘어선 물건들 사이에서 원하는 것을 골랐다. 시장은 언제나 북적인다.

[trade.로쉔]
1 = 로쉔이 귀를 쫑긋 세우며 웃었다. 「좋은 선택이에요. 제 귀가 보증합니다.」
2 = 로쉔의 꼬리가 살랑살랑 흔들린다. 장사가 잘 된 모양이다.

[trade.유린]
1 = 유린이 능글맞게 웃으며 물건을 건넸다. 「이 가격에 이 품질… 여기서만 가능하죠.」
2 = 유린이 코끝을 움찔하며 물건을 감정했다. 「후후, 괜찮은 물건이네.」

[trade.아제르]
1 = 아제르가 계약서를 한 장 내밀었다. 「조건은 명확합니다. 서명을 부탁드립니다.」
2 = 아제르의 차분한 목소리가 조건을 읽어나간다. 까다롭지만 공정한 거래였다.

# ============================================================
# 식사 (eat)
# ============================================================

[eat]
1 = 음식을 한 입 베어 물었다. 따뜻한 기운이 목을 타고 내려간다. 허기가 달래진다.
2 = 조용히 식사를 마쳤다. 속이 든든해지니 마음도 한결 가벼워진다.
3 = 허겁지겁 먹었지만 맛은 느낄 수 있었다. 살아 있다는 건 이런 거다.
4 = 한 숟갈, 한 숟갈 천천히 먹는다. 음식의 온기가 피로한 몸에 스며든다.
5 = 마지막 한 조각까지 깨끗이 먹었다. 남기지 않는 게 음식에 대한 예의다.

[eat.Tavern]
1 = 주점의 따뜻한 식사를 즐겼다. 나무 접시 위의 소박한 요리가 뜻밖에 맛있다.
2 = 모닥불 옆에서 먹는 식사는 어디서든 특별하다. 불빛에 비친 음식이 더 맛있어 보인다.

[eat.Farm]
1 = 갓 수확한 재료로 만든 식사를 먹었다. 신선한 흙 냄새가 아직 남아 있다.
2 = 농장의 그늘 아래에서 식사를 했다. 들판의 바람이 시원하다.

[eat.카엘]
1 = 카엘이 옆에서 같이 먹으며 말했다. 「이거 맛있다! 초원에서 먹으면 더 맛있을 텐데.」
2 = 카엘이 한입에 빵을 우적우적 씹는다. 「배고플 때 먹는 게 제일 맛있어.」

[eat.드라카]
1 = 드라카가 음식을 음미하듯 천천히 먹는다. 「흠, 간이 조금 더 있으면 완벽했을 텐데. 300년의 혀를 믿어.」
2 = 드라카가 눈을 감고 맛을 평가한다. 「나쁘지 않아. 이 마을 요리는 솔직한 맛이라 좋아.」

[eat.브로크]
1 = 브로크가 묵묵히 빵을 씹는다. 한참 뒤에 한마디. 「…든든하군.」

# ============================================================
# 휴식 (rest)
# ============================================================

[rest]
1 = 눈을 감고 숨을 고른다. 긴장이 풀리면서 몸이 조금씩 회복되는 것을 느낀다.
2 = 편한 자리를 찾아 잠시 쉬었다. 짧은 휴식이지만 생각보다 큰 회복이었다.
3 = 등을 기대고 하늘을 바라본다. 구름이 천천히 흘러간다. 마음이 편안해진다.
4 = 팔다리를 뻗고 깊게 쉬었다. 지친 근육이 감사하다는 듯 나른해진다.

[rest.Church]
1 = 교회의 고요한 공기 속에서 쉬었다. 스테인드글라스를 통해 들어오는 빛이 따뜻하다.
2 = 나무 벤치에 앉아 가만히 있었다. 향 냄새와 고요함이 마음의 짐을 덜어준다.

[rest.Lake]
1 = 호수 옆에 누워 물소리를 들으며 쉬었다. 잔잔한 파도 소리가 자장가 같다.
2 = 물가의 바위에 등을 기대었다. 물빛이 반짝이는 것을 바라보다 스르르 눈이 감긴다.

[rest.리엘]
1 = 리엘이 조용히 손을 내밀었다. 따뜻한 빛이 피로한 몸을 감싸는 것 같다. 「좀 나아지셨나요?」
2 = 리엘의 날개가 은은한 빛을 내며 그늘을 만들어 준다. 「천천히 쉬세요. 서두를 필요 없어요.」

[rest.이리엘]
1 = 이리엘이 옆에서 콧노래를 부르며 약초를 다듬고 있다. 편안한 소리다.
2 = 이리엘이 깃털 이불 같은 날개를 살짝 편다. 「바람이 시원하죠? 여기가 쉬기 좋은 자리예요.」

# ============================================================
# 채집 (gather)
# ============================================================

[gather]
1 = 풀 사이를 뒤져 쓸만한 것을 찾았다. 손에 흙이 묻었지만 수확이 있다.
2 = 꼼꼼히 살펴보며 자원을 모았다. 자연이 내어주는 것에 감사한 마음이 든다.
3 = 하나하나 손으로 골라 모았다. 조급하지 않게, 필요한 만큼만.

[gather.Wilderness]
1 = 야생지대의 덤불을 헤치고 약초를 캤다. 야생의 풀은 향이 강하다.
2 = 풀숲 사이에서 반짝이는 것을 발견했다. 야생에는 아직 숨겨진 것들이 많다.

[gather.Herb_Garden]
1 = 정리된 약초밭에서 잘 자란 약초를 따냈다. 은은한 허브 향이 손에 배어든다.
2 = 약초원의 고르게 심어진 줄을 따라 수확했다. 누군가의 정성이 느껴지는 밭이다.

[gather.Mountain_Path]
1 = 바위틈에 끼인 광석을 캐냈다. 산길의 바위는 의외로 많은 것을 품고 있다.
2 = 가파른 비탈에서 조심스럽게 채집했다. 발밑이 불안하지만 보람은 있다.

[gather.이리엘]
1 = 이리엘이 날갯짓으로 바람을 일으키며 약초를 찾아준다. 「이 아이는 상처에 좋은 거예요!」
2 = 이리엘이 높은 곳의 약초를 날아서 따와 건네준다. 「같이 하니까 빠르죠?」

# ============================================================
# 수면 (sleep)
# ============================================================

[sleep]
1 = 눈을 감았다. 어둠 속에서 하루의 기억이 천천히 되감기며, 의식이 멀어진다.
2 = 이불을 끌어당기고 몸을 웅크렸다. 내일은 오늘보다 나은 하루이길.
3 = 깊은 잠에 빠져든다. 꿈속에서 누군가 이름을 부르는 것 같았지만, 돌아보기 전에 잠이 들었다.
4 = 하루의 피로가 눈꺼풀을 무겁게 내리누른다. 이 마을의 밤은 깊고 고요하다.

# ============================================================
# 선물 (gift)
# ============================================================

[gift]
1 = 조심스럽게 선물을 내밀었다. 상대의 표정을 살피며 기다린다.
2 = 주머니에서 꺼낸 것을 건넸다. 받는 이의 반응이 궁금하다.
3 = 말없이 작은 것을 내밀었다. 마음이 전해졌으면 좋겠다.

[gift.loved]
1 = 눈이 반짝이며 양손으로 받았다. 진심으로 기뻐하는 모습에 마음이 따뜻해진다.
2 = 받아든 손이 살짝 떨린다. 이렇게까지 좋아해줄 줄은 몰랐다.
3 = 환하게 웃으며 가슴에 안았다. 그 웃음을 보는 것만으로도 충분했다.

[gift.liked]
1 = 기쁜 표정으로 고개를 끄덕인다. 「고마워요, 마음에 들어요.」
2 = 미소를 지으며 받아들었다. 나쁘지 않은 선택이었던 것 같다.

[gift.disliked]
1 = 잠시 난처한 표정을 짓더니, 어색하게 웃으며 받았다. 취향에는 맞지 않았나 보다.
2 = 고맙다고 말했지만, 시선이 잠깐 어딘가를 헤맸다. 다음엔 더 잘 골라야겠다.

# ============================================================
# 퀘스트 (quest)
# ============================================================

[quest.accept]
1 = 의뢰서에 서명했다. 무거운 책임감과 함께 작은 설렘이 밀려온다.
2 = 퀘스트를 수락했다. 누군가의 부탁을 들어준다는 건, 이 마을의 일원이 된다는 뜻이다.
3 = 의뢰 내용을 다시 한 번 읽어보았다. 간단해 보이지만, 그 뒤에 누군가의 사정이 있을 것이다.

[quest.complete]
1 = 의뢰를 완료했다. 보수보다 더 값진 것은 해냈다는 안도감이었다.
2 = 길드 창구에 완료 보고를 했다. 작은 일이지만, 누군가에게는 큰 도움이었을 것이다.
3 = 끝맺음의 기분은 언제나 특별하다. 오늘도 하나의 이야기를 마무리했다.

# ============================================================
# 활동 (activity)
# ============================================================

[activity.fishing]
1 = 낚싯대를 드리우고 기다린다. 수면에 비친 하늘이 흔들리고, 시간이 느리게 흐른다.
2 = 가벼운 손맛이 전해졌다. 기다림의 보상이 줄 끝에 매달려 올라온다.
3 = 고요한 물가에서의 낚시. 아무것도 잡히지 않아도 이 시간 자체가 쉼이다.

[activity.plant_crop]
1 = 흙에 씨앗을 묻었다. 작은 것 하나를 심는 일이 세상을 가꾸는 일이기도 하다.
2 = 손에 흙을 묻히며 밭을 일궜다. 싹이 돋을 때를 생각하면 기다림도 즐겁다.

[activity.harvest_crop]
1 = 잘 자란 작물을 수확했다. 심었던 씨앗이 이만큼 자랐다니, 시간의 힘을 실감한다.
2 = 두 손 가득 수확물을 안았다. 땅이 내어준 선물에 감사한 마음이 든다.

[activity.heal]
1 = 따뜻한 빛이 상처를 감싸고, 통증이 조금씩 물러간다. 치유란 참을성 있는 일이다.
2 = 기도의 울림이 몸속을 맴돈다. 아픈 곳이 서서히 나아가는 것을 느낀다.

[activity.learn_magic]
1 = 집중하여 마법의 흐름을 느꼈다. 아직 서투르지만, 손끝에서 무언가가 꿈틀거린다.
2 = 마법사의 말을 되새기며 연습했다. 보이지 않는 것을 다루는 일은 언제나 신비롭다.

[activity.mining]
1 = 곡괭이를 내리칠 때마다 암벽이 조금씩 부서진다. 단단한 것 안에 숨겨진 빛을 캐낸다.
2 = 어둠 속에서 광맥의 반짝임을 발견했다. 땀이 흘렀지만 보람 있는 작업이었다.

[activity.hear_rumor]
1 = 흥미로운 소문을 들었다. 진위는 모르겠지만, 이 마을에선 소문도 하나의 이야기다.
2 = 누군가의 속삭임이 귀에 걸렸다. 들은 이야기가 사실이라면… 조금 신경 쓰인다.

[move.Blaze]
1 = 뜨거운 공기가 피부를 달군다. 화염기의 태양 아래, 걸음마다 땀이 흐른다.
2 = 길가의 풀이 바싹 마른 화염기. 그늘이 그리운 길이다.

[move.Frost]
1 = 차가운 바람이 뺨을 스친다. 빙결기의 길은 고요하고 날카롭다.
2 = 서리가 내린 길을 밟는다. 발밑에서 바스락거리는 소리가 청명하다.

[move.Thunder]
1 = 머리카락이 곤두서는 느낌. 뇌명기의 대기는 정전기로 가득하다.
2 = 먼 곳에서 천둥소리가 들린다. 뇌명기의 하늘은 언제나 불안정하다.

[move.Harvest]
1 = 풍요기의 바람이 황금빛 들판을 흔든다. 걷기 좋은 계절이다.
2 = 길가에 열매가 주렁주렁 달려 있다. 풍요기의 혜택이 곳곳에 넘친다.

[move.Radiance]
1 = 광명기의 맑은 햇살이 길을 비춘다. 발걸음이 저절로 가벼워진다.
2 = 투명한 공기 사이로 먼 산까지 보인다. 광명기의 시야는 어디까지나 맑다.

[move.Silence]
1 = 침묵기의 그늘이 길 위에 내려앉아 있다. 발소리조차 조심스러워진다.
2 = 어둑한 공기가 어깨를 짓누른다. 침묵기의 길은 고독하지만, 그래서 생각이 깊어진다.
