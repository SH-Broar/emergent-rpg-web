# ============================================================
# craft_recipes.txt — 제작/합성 레시피 정의
#
# 키:
#   name              — 표시 이름
#   description       — 설명
#   inputs            — 재료 요구. "<itemKey>:<수량>" 을 콤마로 나열.
#                       itemKey 는 ItemType 카테고리 키(Food/Herb/Potion/OreCommon/OreRare/
#                       MonsterLoot/Equipment/GuildCard) 또는 개별 itemId.
#   output            — 결과 산출. "<itemKey>:<수량>" 1건.
#   vigorCost         — 소모 vigor (TP 환산: ceil(vigorCost/10)).
#   requiredLocation  — (선택) 제작 가능 장소. 비우면 어디서나 가능.
#   colorBonus        — (선택) 친화도 보너스 수율 트리거 원소 인덱스. 콤마 구분.
#                       해당 원소 색상값 > 0.6 이면 30% 확률로 +1 추가 산출.
#                       원소 인덱스: 0=Fire 1=Water 2=Electric 3=Iron
#                                    4=Earth 5=Wind 6=Light 7=Dark
# ============================================================
