# Items 1장 즉시 사용 소비품 풀.
# 효과 표기: effects = kind:param:value, kind:value
#   heal:N            HP +N
#   gold:N            골드 +N
#   time-shards:N     시간의 조각 +N
#   color-boost:KEY:N colors[KEY] += N (KEY: fire/water/electric/iron/earth/wind/light/dark)
#   color-all:N       8 컬러 모두 += N
#   grant-card:ID     카드 한 장 컬렉션 추가
#   grant-relic:ID    유물 한 점 추가
#   teleport-village  임의의 village 노드로 즉시 이동 (맵 전용)
#   cleanse-transform 변신(체인지) 정화 원래 모습 복귀
# 전투 포션 (combat = true, 턴당 1회): 전투 중에만 사용. 전투 상태에 직접 작용.
#   combat-mana:N            전투 마나 +N
#   combat-draw:N            전투 중 카드 N장 드로우
#   combat-block:N           전투 중 방어 +N
#   combat-enemy-status:S:N  적에게 status S(vulnerable/weakness/poison 등) N 스택
#   combat-self-status:S:N   자신에게 status S(strength 등) N 스택
#   combat-free-grapple      구속/삼킴 즉시 해제
#   cleanse-group:GROUP      플레이어 디버프를 그룹별 정화 (GROUP: low/mid/high/all). 전투 전용.
#   heal:N (combat=true)     전투 중이면 combat HP + 런 HP 동시 회복
