# Cards MVR 단계 카드 풀.
# 기본 등급 4장 + 일반 5장 + 희귀 2장 + 전설 1장 + 보스 1장 = 13장 원본.
# + 강화판 9장 (기본/일반).
# 효과 표기: effects = kind:value:target, kind:value:target, ...
# upgrade_to = X 공방 강화 시 X 카드로 교체 (없으면 강화 불가).
# 자동 연결: `c-X-plus` 카드가 있으면 로더가 `c-X.upgrade_to`를 자동 설정 (명시값 우선).
#   → 모든 카드는 같은 id에 `-plus` 강화판을 두면 강화 가능. 파일 끝 강화판 섹션 참고.
