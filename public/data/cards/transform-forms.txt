# Transform Forms 변신(체인지/TSF) 폼 전용 카드. (Stage 5)
# source = form → 상점/공방/보상 등 모든 풀에서 *기본* 제외. 변신 중에만 덱에 등장.
# 해제 카드(release-transform)를 쓰면 원래 종족·덱으로 복귀. 안 풀고 이기면 변신이 런에 지속.
#
# 여우 폼(구미호) = Item 37-③ 확장. source=form 이되 *강한 완결 종족급*.
#   - 식별: id 접두 c-fox-* (+ 해제 카드 c-release-change). 변신 중에만 풀 전환(combat-rewards/shop/workshop).
#   - 변신 진입 시 강한 폼 덱(race-form-fox), 변신 중 카드 보상이 여우 풀에서 제시 → 빌드 가능.
#   - baseline 예외(강하게, 도박 페이오프). 적 apply-status는 실효만(vulnerable/weakness/frail/poison/burn).
#   - 컨셉 기둥: 여우불(fire/burn) · 환영(skip/slow) · 홀림(디버프+스케일) · 꼬리(성장) · 변덕(random/amp).
# 신규 핸들러 없음 — 기존 53효과 재사용.
