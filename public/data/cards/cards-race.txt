# Cards 종족 전용 시작 카드 (2026-05-22)
# source=race → 상점/이벤트 풀에서 제외(시작 덱 전용). 각 카드는 -plus 강화판을 가진다(로더 자동 연결).
# 등급별 baseline(basic4/common6/rare9): damage/heal/block 효과는 충족. damage-per-hand 등 측정형은 컨셉 우선.
# 종족 정체성:
#   인간   = 균형(공격+방어 하이브리드, 유연한 드로우)
#   나방   = 속도/연사/드로우(가벼운 회전, 저코스트 연타)
#   팬텀   = 손패 수 관리(damage-per-hand, 패 재배치)
#   아르카나 = 디버프 + 마나(상태이상 누적, 다음 턴 에너지)

# 나방 전용 *바람갈무리* — 시그니처 유물 바람결 깃이 전투 시작마다 손에 1장 지급.
# 0코스트, 2드로우, 사용 후 *손으로 복귀*(버리지 않음) → 덱을 빠르게 훑는 속도 엔진.
# source=form → 어떤 풀에도 등장 안 함, 런 덱 미오염(전투 중에만 존재).
[card.c-moth-windgather]
name = 바람갈무리
rank = common
source = form
element = wind
cost = 0
trigger = manual
effects = draw:2:self, return-self-to-hand
flavor = 바람을 한 줌 쥐었다 펴면, 두 장이 손에 따라 들어온다. 갈무리는 손에 남는다.
